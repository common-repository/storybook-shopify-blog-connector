StoryBook makes it easy to display your existing WordPress blog posts right on your Shopify store. 

== Description ==

StoryBook makes it easy to display your existing WordPress blog posts right on your Shopify store. Once connected, all WordPress posts are instantly available within your store, new posts will be available as soon as you publish.


**NOTE!** - This plugin connects to the "StoryBook ‑ WP Blog Connector" Shopify app.

[Shopify App](https://apps.shopify.com/storybook-wp-blog-connector)
[Support](https://sakuraplugins.com)

### Content type
StoryBook is able to display posts, categories, tags, comments, related, popular and recent posts, and much more. Discover all the available features below.

### Server-Side Caching - Running fast
In order to speed up your store's blog, we are caching the responses on our servers. Caching clearing based on WP entities updates is being handled under the hood, you don't have to do anything. This way your Shopify blog will always be up to date with your WordPress blog and 99% of the time will be cached.

Why is this important? Speed and page response time is the answer. There is a good chance that your Shopify blog is way faster than your original WordPress blog.

### Features
* Two layouts for the posts page template ( Grid and Classic )
* Two sidebars available ( one for single post page and one for posts page )
* Native WordPress support for embeds inside post single like YouTube, Vimeo, Instagram, Facebook, Audio, and others embeds supported via WP Blocks
* Categories page template
* Tags page template
* List comments support
* Recent posts widget
* Popular posts widget
* Categories widget
* Tags widget
* Advertisement banner widget
* Previous/Next Post navigation
* Post excerpt/custom excerpt support
* Yoast SEO meta description support for post single
* Settings & custom CSS style support

### Look & Feel
Your Shopify blog will inherit the styles from the shop's theme, however, in order to make sure everything looks good with any theme, we've also added some CSS styles. Custom CSS is also available within the app's admin area. Need assistance with custom styling? Don't hesitate to contact us.

== Installation ==

This section describes how to install the plugin and get it working. First, you should go and install the ["StoryBook ‑ WP Blog Connector"](https://apps.shopify.com/storybook-wp-blog-connector) Shopify app & follow the instructions.

== Frequently asked questions ==

= Do I also need the StoryBook Shopify app? =

Yes, you do, the app acts as a proxy between your Shopify store and your WordPress blog.

= Is the "StoryBook ‑ WP Blog Connector" Shopify app free? =

No, all blog requests from your Shopify to your WordPress blog go through our servers, therefore, there is a monthly fee.

= What about performance? Page response time. =

We are caching the responses on our servers, there is a good chance that your Shopify blog is way faster than your original WordPress blog.
